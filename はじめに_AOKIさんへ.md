# ネットワーク詳細図ツール ― 使い方ガイド（AOKIさんへ）

ブラウザで開くだけで使えるオンラインツールにしました。**インストール作業は一切不要です。**

**👉 ツール: https://network-design-five.vercel.app/**

---

## このツールでできること

- ノード（人・場所・物・道具・行為）と、それらを結ぶ線（関係性）を、画面上でドラッグ操作で組み立てられる
- 「施主が敷地のアカマツを切って梁にする」みたいな日本語を打ち込むと、AIが自動でノードと線を生成してくれる
- 詳細図 / 全体図 を切り替えて表示できる
- 描いた図はJSONファイルとしてPCに保存・読み込みができる

---

## 最初の1回だけやること（5分）

AI生成機能を使うために、Googleが配布している無料の「APIキー」というパスワードのようなものを取得して、ツールに貼り付けます。**1回貼ればブラウザに記憶されるので、以降は何もしなくてOK。**

### ステップ 1：Gemini API キーを取得

1. ブラウザで **https://aistudio.google.com/** にアクセス
2. Googleアカウントでログイン（普段使っているGmailのアカウントでOK）
3. 左メニューから **「Get API key」** または **「APIキーを取得」** をクリック
4. **「Create API key」** または **「APIキーを作成」** ボタンを押す
5. 表示される長い文字列（`AIza...` で始まる）を **コピー**
   - ⚠️ パスワードと同じ扱い。他人に見せない・SNSに貼らない

### ステップ 2：ツールにAPIキーを貼り付ける

1. **https://network-design-five.vercel.app/** を開く
2. 画面右下の **「✨ AI生成」** ボタンをクリック
3. 「Gemini APIキー」の入力欄に、ステップ1でコピーしたキーを貼り付ける
4. 完了。次回以降は同じブラウザ・同じPCで開けば自動的に認識されます

## 基本の使い方

| やりたいこと | 操作 |
|---|---|
| ノードを追加 | 画面の何もないところを **ダブルクリック** / 上のツールバー「＋ノード追加」 |
| 線（エッジ）を引く | ノードの中心の白い丸から、別のノードへ **ドラッグ** |
| ノードを動かす | ドラッグでそのまま動かせる |
| ノード/エッジを削除 | 選択して **Delete キー** |
| やり直し / 戻す | **Ctrl+Z**（戻す） / **Ctrl+Y**（やり直す）、または上のボタン |
| 詳細図を切り替え | 上のドロップダウンで切り替え |
| 詳細図を追加 | 上の「＋詳細図」ボタン |
| 自動配置 | 上の「⚙ 自動配置」ボタン（力学レイアウト） |
| AIで自動生成 | 画面右下の **「✨ AI生成」** をクリック |

### AI生成の使い方

1. 右下の「✨ AI生成」を開く
2. テキスト欄に日本語で関係性を書く
   - 例：「施主が敷地のアカマツを切り、敷地内で乾燥させ、チョウナではつって、家の梁として使う。」
3. 「ノード/エッジを生成」ボタンを押す
4. 数秒待つと、ノードと線が自動で増える

⚠️ 既存のノードに追加されます。重複は自動マージされないので、似た名前のノードができてしまったら手動で削除してください。

## 大事：作った図の保存

このツールは **自動保存しません**。ブラウザを閉じる前に、必ずJSON保存してください。

### 保存
- 上のツールバー **「JSON保存」** ボタン → JSONファイルがPCにダウンロードされます
- 名前をつけて分かりやすい場所に置いてください（例: `デスクトップ/安曇野_工房.json`）

### 読み込み（次回以降）
- ツールを開く → 上の **「JSON読込」** → 保存したファイルを選択 → 続きから編集できる

### クリア
- 全部消したい時は上の **「クリア」** ボタン（戻すボタンで復元できます）

## 困ったとき

### AI生成でエラーが出る
→ APIキーが間違っているか、Google側でAPIが有効化されていない可能性。
Google AI Studio (https://aistudio.google.com/) でキーを再発行して、右下のAI生成パネルで貼り直してみてください。

### APIキーをやり直したい / 違うキーに変えたい
→ ブラウザのキャッシュをクリアするか、開発者ツール（F12）→ Application → Local Storage で `vd_gemini_api_key` を削除 → ページ再読み込みすると、入力欄が再表示されます。

### 画面が真っ白 / 動かない
→ ブラウザを **Chrome** または **Edge** の最新版で開いてみてください。Safari でも動くはずですが、未検証です。

### スマホ・タブレットで使いたい
→ 一応開けますが、ドラッグ操作前提なのでPC推奨です。
